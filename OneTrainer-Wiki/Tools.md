## 视频工具

这提供了一些基本工具，用于从视频中获取截图、将长视频分割成短片以及从 url 下载视频。对于某些视频格式，可能需要安装 [ffmpeg](https://www.ffmpeg.org/)。

多个视频是并行处理的，如果您有一个非常长的（如电影长度的）视频，那么手动分割成几个较短的块会运行得更快。

**提取片段** - 指定单个视频文件或包含多个视频的文件夹（包括子文件夹中的）以及输出文件夹的路径。如果启用了"输出到子目录"选项，输出将保存到输出文件夹下为每个处理的视频单独的文件夹中，否则它们都将保存到输出文件夹的顶层。视频保存为 .avi 格式。

如果在"时间范围"中指定了开始和结束时间，则提取的片段将限制在该范围内。这仅适用于单视频处理，从多个视频的目录中提取时将忽略该范围。时间可以以"hh:mm:ss"、"mm:ss"或"ss"格式指定。

如果启用了"在剪辑处分割"选项，它将使用 [PySceneDetect](https://www.scenedetect.com/) 来识别视频中的剪辑并在这些位置分割输入视频。否则分割可能发生在任何位置，输出片段可能包含剪辑。

"最大长度"设置确定输出片段的长度，以秒为单位。使用"在剪辑处分割"选项检测到的场景如果较长，可能会被分割成较短的部分。如果场景很短，结果片段可能比指定的短，例如一个 4 秒的场景，最大长度为 3 秒，将被分割成 2 秒的片段。任何 <0.25 秒的部分都将被丢弃。

"设置 FPS"将重新编码输出视频到指定的 FPS。大多数视频模型都有它们最适合的固定帧率，不同帧率的视频可能会导致输出看起来太快/太慢，所以这可以帮助防止这种情况。Hunyuan 使用 24 fps，这是默认值。设置为 0 禁用此功能，并以与源视频相同的帧率输出视频。注意，您的系统上需要安装 FFmpeg - 在 Linux 上您应该能够执行 ```apt install ffmpeg``` 或发行版的等效命令，对于 Windows [需要更多步骤](https://www.wikihow.com/Install-FFmpeg-on-Windows)。

选择"移除边框"将自动剪掉视频周围的黑条，这在信箱式视频中很常见。如果视频的某个部分主要是黑色的（因此无法与填充区分开），或者如果黑条中存在水印或徽标，这可能无法按预期工作。

"裁剪变化"将随机改变输出片段的宽高比，具有可配置的变化，这可能有助于避免在训练期间过拟合到特定的宽高比，但可能会有裁剪掉边缘细节的风险，因此应审查输出片段以确保它们与描述匹配。片段将被裁剪为围绕原始值的随机宽高比（定义为高度/宽度，所以 3:2 的视频将是 0.66），略微偏向正方形裁剪，裁剪中心沿裁剪轴随机选择。初始宽高比接近正方形（1）的片段将在 aspect-var 和 aspect+var 之间输出，横向视频（<0.85）将在 aspect*(1-var/2) 和 aspect*(1+var*1.5) 之间裁剪，纵向视频（>1.2）在 aspect*(1-var*1.5) 和 aspect*(1+var/2) 之间。

**提取图像** - 与片段相同，指定输入视频/文件夹和输出文件夹，从单个视频提取时可配置时间范围，并可选择输出到子目录。图像保存为 jpeg 格式。

指定每秒捕获的图像数量，默认是 0.5，即每 2 秒一张图像。图像以该频率在"中心"帧周围随机分布选择，如果运行多次，您最终会选择不同的帧。

"模糊移除"指定一部分捕获的图像作为"太模糊"而丢弃，这可能有助于避免捕获处于过渡中或有运动模糊的帧。使用 ["拉普拉斯方差"](https://pyimagesearch.com/2015/09/07/blur-detection-with-opencv/) 方法来量化图像的清晰度，并丢弃所选帧集中最低的，只保存其余的。

"裁剪变化"的工作方式与上面针对片段描述的相同。

**下载** - 提供单个链接或链接列表（.txt 文件，每个链接占一行）以下载到指定的输出目录。使用 [yt-dlp](https://github.com/yt-dlp/yt-dlp)，它支持各种各样的网站。可以提供额外的参数，请参阅 yt-dlp github 页面获取适用参数和支持网站的列表。默认参数只是在显示下载进度时减少终端窗口中的垃圾信息。
