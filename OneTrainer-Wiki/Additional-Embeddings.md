# 描述
通常被称为"关键调优"，这是一种将文本嵌入添加到任何训练中的技术。这不限于 LoRA 训练，还可以与微调一起使用，甚至可以只训练多个嵌入而不训练任何 Unet、DIT 或文本编码器。

无论您进行什么类型的训练，额外嵌入选项卡始终可用。这就是您可以将额外嵌入与 LoRA 训练或微调结合使用的方式。

有两种使用额外嵌入的技术。

1. 关键调优 - 单独训练一个嵌入，并使用额外嵌入选项卡加载输出的 safetensor 用于您的 Lora 或微调训练。
    1. 这种方法的好处是，完成后，您有一个固定的目标来训练 unet 或 DIT。
    2. 缺点是您需要先训练嵌入。虽然嵌入训练很快，但仍然需要时间和管理。
2. 组合训练 - 在您进行 lora 或微调的同时训练额外嵌入
    1. 这种方法的好处是您不需要先训练嵌入
    2. 这种方法的缺点是，对于您的 unet 或 DIT 训练工作来说，嵌入是一个移动目标。最好在您的 unet 或 DIT 之前停止训练嵌入，以允许训练在嵌入停止的最终区域进行。

# 用法
根据您训练的内容，额外嵌入的输出会有所不同。

训练 Lora 时，默认是将嵌入打包为 Lora 的一部分。如果您不想要这样，请取消选中 Lora 选项卡上的框。

只训练嵌入或微调时，会创建一个子文件夹，里面有嵌入。此子文件夹将位于您完成的 safetensor 文件存储的位置。

这也适用于训练期间的增量保存，将为每个增量保存创建带有嵌入的子文件夹（或打包在 Lora 中）。

并非所有生成程序都适用于与 Lora 打包的嵌入，所以如果您不想要此功能，请禁用它。

要与生成一起使用，您需要将 LoRA 和嵌入，或微调和嵌入，包含到您选择的生成程序中以获得组合结果。

# 额外嵌入 GUI 设置
## 添加嵌入
* 按下此按钮将向选项卡添加一个默认嵌入。

## 额外嵌入设置
* 红色 X - 按下此按钮将删除此额外嵌入
* 绿色加号 - 按下此按钮将复制您的额外嵌入
* 基础嵌入（默认：空白）- 使用此字段指定是否要加载现有嵌入以在训练期间使用。它不需要再训练。
* 占位符（默认：`<embedding>`）- 您的嵌入的占位符。它将用作文件名（在模型选项卡中定义的模型旁边的单独文件夹中生成），并且必须与您在描述中使用的触发词匹配。这也将是您在提示词中使用的占位符，例如在 Automatic1111 中。注意，它不需要是单个单词，用空格分隔的几个单词也可以。
* token 数量（默认：1）- 您想要用于嵌入的 token 数量。如果您将此字段留空，OneTrainer 将使用您的初始嵌入文本来自动计算 token 数量。如果您设置一个值，它将用 `*` 补全缺失的 token（例如：token 数量 3 和 dog 作为初始文本，这是一个 token，将变成 `dog**`）。
* 训练（默认：开启）- 一个开关，指定您的下一次训练运行是否将训练此额外嵌入
* 输出嵌入（默认：关闭）- 一个开关，指定您是否希望嵌入在文本编码器的输出而不是输入上训练。此选项旨在帮助使用 T5 或其他 LLM 的模型。
* 之后停止训练（默认：空白 - 从不）- 两个字段，您可以用来告诉 OneTrainer 您想要训练此嵌入多长时间。指定从不将允许训练运行整个长度。
* 初始嵌入文本（默认：* ）- 您的嵌入在任何训练发生之前最初指向的单词或单词或短语。如果您在这里放了太多 token，它将被截断到 token 数量，如果您指定太少，它将用 * 填充。

# 注意事项
* 使用像 automatic1111 或 stable swarm 这样的工具来确定您想要使用的初始文本有多少 token。这可以帮助设置您想要在嵌入中使用的 token 数量。您也可以使用此 [链接](https://novelai.net/tokenizer) 来确定 token 信息。请确保您从下拉列表中使用 CLIP Tokenizer。
    * 或者，您现在可以将 token 数量留空，OneTrainer 将为您计算
* 将额外嵌入与同时训练 unet 和文本编码器的 LoRA 训练一起使用，在主体训练的情况下会导致非常快速的学习
* 由于额外嵌入目前是单独的文件，尝试训练非常大量的嵌入不容易管理或创建。
* 可能还有更多关于可以用这种技术做什么的想法。试试看，并在 Discord 上分享！
* 使用您的描述触发词作为嵌入的占位符会使事情变得容易得多。您不再需要使用 `<embedding>` 并为嵌入运行和 LoRA/微调/额外嵌入运行使用单独的描述。
* 如果您只训练 unet 和额外嵌入（并且不训练文本编码器），Prodigy 会有困难。将增长限制为 2，并使用 BF16 权重和计算已被证明有效，至少在 SDXL 中是这样。如果您尝试使用 FP16 的 Prodigy 并使其工作，请分享您的设置。
* 在训练选项卡的嵌入学习率下，有一个选项"保留嵌入范数"，它用于将训练后的嵌入重新缩放到所有其他嵌入的平均范数（在 0.35 和 0.45 之间）。
* LR 可能需要适应每个编码器，但实际上没有任何可以给出的一般指导，因为这在很大程度上取决于您在训练什么。
