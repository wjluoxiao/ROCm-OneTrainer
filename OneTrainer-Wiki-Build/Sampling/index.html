    采样选项卡

    采样选项卡是您可以控制要创建什么样本以及何时创建的地方。

    在主选项卡上，您可以执行以下操作：

    顶部项目

    • 每 X Y 采样一次。其中，x 是整数，y 可以是以下任意一种：Epoch/Step/Second/Minute/Hour/Never/Always。注意，Never 和 Always 是特殊的，会忽略整数输入。
    • 选择图片格式。默认是 JPG。也可以使用 PNG
    • 立即采样。当您按下此按钮时，OneTrainer 将在下一个可用机会进行采样
    • 手动采样。按下此按钮将打开手动采样窗口，请参阅下面的更多信息。
    • 非 EMA 采样。启用此开关将允许在使用 EMA 时生成 EMA 和非 EMA 采样。
    • 样本到 Tensorboard。启用此开关将允许样本作为 Tensorboard 运行数据的一部分加载。

    样本管理

    • 下拉菜单将显示您已创建的样本配置
    • 添加配置按钮将允许您创建新的样本配置文件
    • 添加样本按钮将在下面的空间中创建一个新样本
    • 注意：一种轻松将样本从一个配置复制到另一个配置的方法是转到您的 training_samples 文件夹并找到您想要复制的 .json 文件。复制它并将其重命名为您想要的配置名称。然后您可以在 OneTrainer 中加载这个新配置并修改它。或者可以在您选择的文本编辑器中修改。

    样本提示词

    • 在这个区域，样本提示词将出现，您可以在这里对它们进行一些高级修改。
    • 要查看所有选项，您必须按下样本右侧的 ... 按钮。
    • 您可以使用样本前面的按钮轻松删除或复制样本。
    • 样本前面的开关控制样本是否启用。注意：无法使用 ... 按钮编辑未启用的样本。

    样本提示词配置窗口

    在按下 ... 按钮后看到的这个窗口中，您可以对与样本提示词相关的所有项目进行更改。这包括： * 提示词 - 您可以编辑主提示词。 * 负面提示词 - 您可以编辑样本的负面提示词。 * 宽度和高度 - 您可以编辑提示词的宽度和高度。注意：这些值必须对您正在使用的模型有效。默认是 512x512。 * 种子 - 您可以输入您想要使用的种子。默认值是 42，因为毕竟它是答案。 * 随机种子 - 启用此开关将覆盖您的种子并改用随机种子。 * cfg 比例 - 用于提示词的配置比例，默认是 7.0 * 采样器 - 您想要用于样本提示词的采样器。默认是 DDIM。选项包括 Euler/Euler A/UniPC/Euler Karras/DPM++ Karras/DPM++ SDE Karras * 步骤 - 您想要用于样本的步骤数。默认是 20。

    手动采样窗口

    按下手动采样时打开的这个窗口将允许您在需要时生成自定义手动样本。这对于微调和其他具有多个概念的长训练运行非常有用。 此设置与样本提示词配置窗口相同，但仅供一次性使用。按下采样将导致 OneTrainer 在可能时进行采样。

    注意事项和评论

    • 此选项卡可以在训练期间更新，并在您进行更改时更新您的样本和频率。如果您开始设置下一次运行的样本，而当前训练开始样本生成运行，这可能会导致无人看管的后果。
    • 嵌入的样本必须使用嵌入占位符才能工作。
    • 样本在创建时会放入您的工作区目录的样本文件夹中。
    • 样本放入子文件夹，基于样本顺序（0,1,2）后跟提示词的第一部分。注意：更改提示词的开头将更改该步骤之后样本将进入的文件夹。
    • 手动样本进入它们自己的子文件夹。
    • Hunyuan 视频采样注意 - 超过 75 个 token 或接近 75 个 token 的提示词可能会导致样本出现像素化和失真。建议在训练此模型时保持提示词较短。
